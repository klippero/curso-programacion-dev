Implementa la clase ListaPalabras que se construya a partir de una lista de palabras y tenga un método first_and_last que devuelva el número de palabras de 2 o más letras que tengan la primera y la última letras iguales.

Las cadenas de caracteres se consideran listas por lo que el acceso a las letras de la cadena se realizan igual que el acceso a una lista. De esta forma, el primer elemento se accede mediante [0] y el último mediante [-1].
