Toma la clase CajaRegistradora y modificala para que las instancias se guarden en un fichero binario.

En el constructor se debe recibir el nombre de la caja registradora que será el nombre del fichero.

Implementa el método save que vuelca el objeto al fichero.

Además modifica el constructor para que, si existe el fichero, se lea el contenido del mismo.

Ejecuta la prueba sin tener ningún fichero de datos y verás que se crea uno llamado cr1.dat.
