Las excepciones se pueden controlar, esto es, no necesariamente tienen que interrumpir la ejecución del programa. Para controlar las excepciones lo que se hace es capturarlas.






Cuando se produce una excepción dentro del código a controlar, se interrumpe la ejecución de ese bloque de código pero la ejecución continúa en el código a ejecutar en caso de excepción.



En el siguiente ejemplo se pide al usuario un número que se usa para dividir 100. Si el número introducido es 0 se producirá una excepción que es capturada para evitar que se interrumpa la ejecución y mostrar el mensaje de error no se puede dividir entre 0
