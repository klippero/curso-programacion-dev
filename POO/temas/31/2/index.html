Las excepciones se pueden controlar, esto es, no necesariamente tienen que interrumpir la ejecución del programa. Para controlar las excepciones lo que se hace es capturarlas.






Cuando se produce una excepción dentro del código a controlar, se interrumpe la ejecución de ese bloque de código pero la ejecución continúa en el código de rescate.



En el siguiente ejemplo se pide al usuario un número que se usa para dividir 100. Si se introduce el 4, se muestra el cociente es 25. Si volvemos a ejecutarlo y se introduce el 0, se produce una excepción que interrumpe la ejecución del bloque (no se muestra el cociente) y se rescata la ejecución continuando en el bloque de rescate y mostrando no se puede dividir entre 0.
