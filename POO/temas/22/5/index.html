En aquellos casos donde se esté buscando un elemento no será necesario recorrer todo el array sino que la búsqueda concluirá cuando se localice dicho elemento.

Para recorrer un array utilizando un bucle while procedemos de la siguiente forma:

  1. inicializamos el contador al primer elemento que es el 0
  2. iteramos mientras el contador sea menor que el tamaño del array
  3. al finalizar el bucle incrementamos en 1 el contador
