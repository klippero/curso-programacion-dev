Cuando se intenta abrir un fichero en modo lectura, se producirá un error si este no existe. Por ello, es importante chequear previamente si el fichero existe.

No ocurre lo mismo cuando se abre el fichero para escribir o añadir. En estos casos, si el fichero no existe, se crea.

En los ejemplos anteriores, si el fichero DATA_FILE (donde se guarda la temperatura) no existe se produce un error. No ocurre lo mismo con el fichero de registro de temperaturas, que, en caso de no existir se crea.

El ejemplo comprueba si existe el fichero y, en caso de no existir, usa la temperatura de 0ºC.
