En una primera aproximación a los ficheros vamos a ver cómo leer todo el contenido de un fichero con una única sentencia.

La secuencia de acciones que se realizan son:

  1. Abrir el fichero
  2. Leer el contenido
  3. Cerrar el fichero

En el constructor del siguiente ejemplo se lee el contenido del fichero 30.1.txt que contiene la temperatura del frigorífico. Si el fichero no existe se producirá un error.

Antes de probar el código crea el fichero 30.1.txt con el valor 15.
