Para añadir contenido al final de un fichero bastará con indicarlo al abrirlo.

La secuencia de acciones es:

  1. Abrir el fichero para añadir
  2. Escribir el contenido (automáticamente se añade al final)
  3. Cerrar el fichero

Es fundamental cerrar siempre el fichero después de ser usado para evitar comportamientos no deseados.

El método fija_temperatura, además de guardar la nueva temperatura en el fichero, la añadirá en la última línea del fichero de registro de temperaturas. Si el fichero no existiese, se crea.

Antes de probar el código crea el fichero 30.3.txt con el valor 15.

Después de ejecutarlo verás que se ha creado el fichero 30.3.reg.txt con todas las temperaturas que se han cambiado, una en cada línea.
